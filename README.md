# AutoSlab

Fabric-мод для Minecraft **1.21.11**. Зажимаешь клавишу — пока держишь полублоки
в руке, мод сам сканирует область вокруг тебя и ставит полублоки на все полные
блоки поблизости (только если сверху пусто).

## Настройка

### Через Mod Menu (рекомендуется)

Если установлены **Mod Menu** и **Cloth Config API**, у AutoSlab в списке модов
появится шестерёнка настроек — там же на одном экране: вкл/выкл, слайдер
интервала, слайдер радиуса и поле для клавиши (кликни по нему и нажми нужную
кнопку, как в любом другом моде с GUI-настройками).

- **Mod Menu** — https://modrinth.com/mod/modmenu (для 1.21.11: v17.0.0)
- **Cloth Config API** — https://modrinth.com/mod/cloth-config (для 1.21.11: v21.11.153, обязателен, без него экран настроек не откроется)

Cloth Config зашит в `depends` мода, так что без него AutoSlab вообще не
запустится — положи его jar рядом в `mods`. Mod Menu опционален: без него
мод продолжит работать, просто не будет шестерёнки в списке модов (тогда
настройки — через клавиатуру/команды ниже).

### Без Mod Menu

- **Клавиша**: `Настройки → Управление (Controls) → категория "AutoSlab"`.
  По умолчанию клавиша не привязана — назначь любую свободную.
- **Интервал и радиус** — через команду в игровом чате:
  - `/autoslab toggle` — включить/выключить мод
  - `/autoslab interval <тики>` — сколько тиков ждать между расстановками (20 тиков = 1 сек), 1–200
  - `/autoslab radius <блоки>` — радиус поиска полных блоков вокруг игрока, 1–8
  - `/autoslab status` — показать текущие настройки

Все настройки (независимо от способа изменения) сохраняются в `config/autoslab.json`.

## Сборка через GitHub

1. Создай новый репозиторий на GitHub и залей туда содержимое этой папки
   (или распакуй архив прямо в корень нового репозитория).
2. Зайди во вкладку **Actions** — workflow `Build AutoSlab` запустится
   автоматически при пуше в ветку `main` (или запусти вручную кнопкой
   **Run workflow**, т.к. добавлен `workflow_dispatch`).
3. После завершения сборки открой запущенный workflow → внизу будет
   артефакт **autoslab-mod** с собранным `.jar`.
4. Скачай `.jar`, положи в папку `mods` клиента с установленным
   **Fabric Loader 0.18.1+** и **Fabric API** для 1.21.11.

Локально мод собирается командой `./gradlew build` (потребуется сгенерировать
Gradle wrapper через `gradle wrapper` при первой локальной сборке, либо
использовать установленный Gradle 8.11+ напрямую: `gradle build`).

## Возможные доработки

- Поддержка "долива" верхней половины двойного слэба, если внизу уже
  стоит полублок того же материала.
- Фильтр по конкретному виду блока (ставить только на определённые полные блоки).
